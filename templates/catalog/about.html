About This Project

TV Shows Catalog is a Django-based web application built to practice back-end development, data handling, and dynamic user interaction. The project focuses on clean structure, filtering logic, and improving the browsing experience through responsive updates.

Technologies Used

  • Django (Back-end & ORM)
  • Bootstrap 5 (Green Template)
  • JavaScript (AJAX for dynamic filtering)

Key Features

  • Dynamic filtering by genre and country
  • Search functionality
  • State persistence via URL parameters
  • AJAX updates without full-page reload
  • Data cleanup and normalization to ensure consistent filtering
